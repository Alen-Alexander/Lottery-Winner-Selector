# Lottery-Winner-Selector
 Interactive Bootstrap lottery winner selector with random ticket generation, modal popup, and real-time result display. JS + Bootstrap demo.

Features
- **Random Selection**: Picks from predefined ticket array `[101,202,303,404,505]`
- **Bootstrap Modal**: Professional winner announcement popup
- **Real-time Display**: Updates page + modal simultaneously
- **Console Logging**: Debug-ready (`console.log(winner)`)
- **Bootstrap 4 Integration**: Full CDN stack (CSS + jQuery + Popper + JS)
